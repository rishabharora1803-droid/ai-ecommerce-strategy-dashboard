# AI-Integrated E-Commerce Strategy Dashboard  
**Strategic Management (SM) Project**

---

## 1. Project Overview
This project presents an AI-integrated strategic analysis of the Indian e-commerce industry, with a primary focus on **Amazon India** and benchmark comparison with **Flipkart and Meesho**.  
The objective is to apply AI-driven analytics and strategic frameworks to support managerial decision-making and strategic positioning.

---

## 2. Repository Contents
This repository contains the complete artefacts required to review and run the interactive dashboard:

```
├── SM_Code.py / app.py        # Main dashboard application
├── ecommerce_data.csv         # Dataset used for analysis
├── README.md                  # Project documentation
```

## 3. Nature of the Dashboard
The dashboard is **interactive in nature** and enables:
- Executive KPI analysis
- Strategic group positioning
- Trend forecasting
- Scenario-based strategic decision simulation

The focus is on **strategic insight and decision evaluation**, not only descriptive analytics.

## 4. How Faculty Can Access the Interactive Dashboard

### Option A: Run the Dashboard Locally (Recommended)
The dashboard can be accessed live by running it locally using Python.

#### Step 1: Install Python
Ensure Python 3.9 or above is installed.
Check using:
```bash
python --version
```

#### Step 2: Install Required Libraries
Run the following command in the project folder:
```bash
pip install streamlit pandas numpy scikit-learn plotly
```

#### Step 3: Launch the Dashboard
Execute:
```bash
streamlit run SM_Code.py
```
(or)
```bash
streamlit run app.py
```

#### Step 4: Open in Browser
Once launched, the dashboard opens at:
```
http://localhost:8501
```

The dashboard is fully interactive with tabs, filters, and scenario controls.
